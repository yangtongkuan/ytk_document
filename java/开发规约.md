### 开发规约

#### 1. 所有的集合类不允许返回为null

| 项目 | 方法                        | 值                         |
| ---- | --------------------------- | -------------------------- |
| List | **Collections.emptyList()** | **Collections.EMPTY_LIST** |
| Map  | **Collections.emptyMap()**  | **Collections.EMPTY_MAP**  |
| Set  | **Collections.enptySet()**  | **Collections.EMPTY_SET**  |
